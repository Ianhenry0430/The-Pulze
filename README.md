# 🏆 The Pulze

> Your pulse on sports. Live standings, real stats, and the news that matters.

The Pulze is a sports hub for fans who want everything in one place —
live standings, player and team stats, and breaking news across all major leagues.
No fluff. Just sports.

---

## ⚡ Features

- 📰 **Sports News Feed** – Breaking news and headlines across NFL, NBA, MLB, NHL, and more
- 🏅 **Live Standings** – Up-to-date league and division standings updated in real time
- 📊 **Stats Center** – Player and team stats with filters by season, league, and position
- 🔔 **Game Alerts** – Get notified when your team plays, scores, or wins
- 🎮 **Pulze Game** – Daily sports trivia and prediction challenges against other fans
- ❤️ **Follow Your Teams** – Personalize your feed around the teams and leagues you care about
- 🏆 **Leaderboard** – Compete with other fans on prediction accuracy and trivia scores

---

## 🛠️ Tech Stack

| Layer | Tech |
|---|---|
| Frontend | React + Tailwind CSS |
| Backend | Node.js + Express |
| Database | PostgreSQL + Prisma ORM |
| Auth | Clerk |
| Sports Data | ESPN API / SportsData.io |
| Real-time | Socket.io |
| Hosting | Vercel (frontend) + Railway (backend) |

## 🚀 Getting Started

### Prerequisites

- Node.js v18+
- PostgreSQL
- A [SportsData.io](https://sportsdata.io) API key

### Installation

```bash
# Clone the repo
git clone [github.com](https://github.com/Ianhenry0430/The-Pulze.git)
cd The-Pulze

# Install dependencies
cd client && npm install
cd ../server && npm install
